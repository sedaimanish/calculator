# Calculator

A simple calculator built with React and CSS.

## Features

* Basic arithmetic operations

  * Addition (+)
  * Subtraction (-)
  * Multiplication (×)
  * Division (÷)
* Clear display
* Responsive design
* User-friendly interface

## Technologies Used

* React
* JavaScript
* CSS

## What I Learned

Through this project, I practiced:

* React components
* State management with hooks
* Event handling
* CSS styling
* Building interactive user interfaces

## Live Preview

https://calculator-sedaimanish.vercel.app/

## Getting Started

1. Clone the repository
2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm start
```
